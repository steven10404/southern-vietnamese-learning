# GitHub Pages

This repository can be served as a static GitHub Pages site from the `main` branch root.

Pages URL:

```text
https://steven10404.github.io/southern-vietnamese-learning/
```

GitHub Pages can run the learning UI, article/phrase/vocabulary pages, local JSON data, and the browser-side RAG search.

GitHub Pages cannot run the local Node/VieNeu backend, so VieNeu-TTS stays available only when opening the local app through:

```text
http://localhost:5178
```
